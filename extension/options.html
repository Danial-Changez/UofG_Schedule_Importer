How to use Schedule Exporter

This extension extracts your course schedule from the university timetable page and exports it to calendar services (Google/Outlook).

  1. Open your timetable page and run the extension's popup.
  2. Select the term(s) and choose 'Export' or 'Import'.
  3. For Google imports the extension will request consent via Chrome's account picker.
  4. Use the Save button on this page to persist OAuth client IDs (optional; usually not required when using chrome.identity).

If you need more help, see the README or contact the developer.
